🐦 Flappy Bird - JavaScript Puro

Projeto desenvolvido com foco na prática de lógica de programação e manipulação avançada do DOM, recriando a mecânica clássica do jogo Flappy Bird utilizando apenas tecnologias nativas do navegador.

🚀 Sobre

Este projeto consiste em um clone funcional do Flappy Bird desenvolvido com JavaScript puro (Vanilla JS), HTML e CSS, sem o uso de frameworks ou bibliotecas externas.

O jogo foi estruturado utilizando funções construtoras e organização modular, simulando física básica, geração dinâmica de obstáculos e sistema de pontuação em tempo real.

🧠 Conceitos Aplicados

Manipulação dinâmica do DOM

Programação orientada a objetos com funções construtoras

Controle de animação com setInterval

Reaproveitamento inteligente de elementos

Cálculo de posicionamento com clientWidth e clientHeight

Detecção de colisão com getBoundingClientRect

Controle de estado do jogo (Game Loop)

🧰 Tecnologias Utilizadas

HTML5

CSS3

JavaScript (ES6+)

🎮 Funcionalidades

Sistema de voo com gravidade e impulso

Geração dinâmica de barreiras

Sistema de pontuação automática

Detecção de colisão

Encerramento do jogo ao colidir

Estrutura organizada para fácil expansão

🎯 Objetivo

O principal objetivo deste projeto foi fortalecer fundamentos sólidos em JavaScript e compreender como estruturar aplicações interativas sem o uso de frameworks.

Além disso, o projeto simula a arquitetura básica de jogos 2D no navegador, reforçando conceitos essenciais para desenvolvimento front-end moderno.

📌 Aprendizados

Durante o desenvolvimento foram reforçados:

Organização e separação de responsabilidades

Estruturação de lógica orientada a objetos

Controle manual de animações

Pensamento algorítmico aplicado a problemas visuais
