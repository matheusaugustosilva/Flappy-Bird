# 🐦 Flappy Bird | JavaScript Puro

> Clone do clássico Flappy Bird desenvolvido do zero utilizando apenas tecnologias nativas do navegador.

---

## 🌟 Visão Geral

Este projeto recria a mecânica do Flappy Bird aplicando conceitos fundamentais de desenvolvimento front-end.
Toda a lógica foi construída com **JavaScript puro (Vanilla JS)**, sem uso de frameworks ou bibliotecas externas.

O foco principal foi consolidar fundamentos sólidos de programação e estruturação de aplicações interativas.

---

## 🧠 Conceitos Trabalhados

* Manipulação dinâmica do DOM
* Programação orientada a objetos com funções construtoras
* Controle manual de animação (`setInterval`)
* Sistema de física simples (gravidade e impulso)
* Reaproveitamento de obstáculos para otimização
* Detecção de colisão com `getBoundingClientRect`
* Controle de estado e loop principal do jogo

---

## 🎮 Funcionalidades

✔️ Movimento contínuo das barreiras
✔️ Geração dinâmica de aberturas aleatórias
✔️ Sistema de pontuação em tempo real
✔️ Detecção de colisão precisa
✔️ Encerramento automático ao colidir

## 🏗️ Estrutura do Projeto

O código foi organizado em componentes independentes:

* `Passaro` → Controle de movimento e física
* `Barreira` → Estrutura individual das barreiras
* `ParDeBarreiras` → Controle de obstáculos superiores e inferiores
* `Barreiras` → Gerenciamento de múltiplos obstáculos
* `Progresso` → Sistema de pontuação
* `FlappyBird` → Loop principal do jogo

Essa organização facilita manutenção e futuras melhorias.
